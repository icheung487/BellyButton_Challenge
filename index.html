Belly Button Biodiversity Dashboard

Project Overview

This project consisted on creating a dashboard for users to visualize the bacterial data for each volunteer. This is an interactive dashboard that will identify the top 10 bacterial species in their belly buttons. This is a great way to visualize whether specific species is found in their navel.

Instructions

Click on your ID number in the Subject ID filter box to see your profile and bacterial data.

Demographic Info

The bar chart displays the top 10 bacterial species(OTUs) when a volunteer filter their own ID number.

The gauge chart displays the weekly washing frequency's value, showing the value as a measure base on volunteer's ID.

The bubble chart will display the "otu_ids" and "sample_value." When you hover over a bubble it will display the name of the bacteris when looking at which bacteria sample is large or small. Please note the chart only displays when an individual filters their ID number.
